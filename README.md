# Финансовый автопилот PWA

Локальное PWA-приложение для iPhone: памятка распределения аванса и зарплаты.

## Важно про приватность

- В коде нет личных финансовых данных.
- Все введённые суммы сохраняются локально в браузере устройства через `localStorage`.
- По ссылке другой человек увидит пустое приложение и свои локальные данные, а не ваши.
- PIN защищает от случайного просмотра. Для более сильной защиты нужно добавить локальное шифрование.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация

Проще всего развернуть через Vercel или Netlify, подключив приватный GitHub-репозиторий.
